# 拼音学习乐园 🎓

一个面向幼儿园的拼音学习网站，帮助小朋友快乐学习汉字拼音。

## ✨ 功能特点

- 🎯 **考试功能**：每次20道题，随机从题库抽取
- 📝 **拼音选择**：每个汉字提供4个拼音选项（带音调）
- ✅ **即时反馈**：答对自动进入下一题，答错显示正确答案
- 📊 **成绩统计**：考试结束后显示分数和错题详情
- 💾 **历史记录**：使用 IndexedDB 本地存储所有考试记录
- 📱 **响应式设计**：完美适配 Pad 和手机端
- 🎨 **卡通风格**：鲜艳的色彩和友好的界面设计

## 🛠️ 技术栈

- **Vue 3** - Composition API
- **TypeScript** - 类型安全
- **Vue Router 4** - 路由管理
- **Pinia** - 状态管理
- **UnoCSS** - 原子化 CSS
- **IndexedDB** - 本地数据存储（通过 idb 库）
- **Vite** - 构建工具

## 🎮 使用说明

1. **开始考试**：点击首页的"开始考试"按钮
2. **答题**：查看汉字，从4个选项中选择正确的拼音
3. **查看结果**：完成20题后查看分数和错题
4. **历史记录**：点击右上角"历史考试"查看所有考试记录

## 🎨 设计特色

- **卡通配色**：明亮的橙色、绿色、蓝色等儿童友好色彩
- **大字体**：适合儿童阅读的大号字体
- **动画效果**：有趣的动画提升学习体验

## 📝 题库说明

当前包含 50 个基础常用汉字，如：
- 人、口、手、大、小
- 上、下、左、右、中
- 天、地、日、月、水、火
- 爸、妈、哥、姐、弟、妹
- 等等...

每个汉字都配有4个相似的拼音选项（音调不同），帮助孩子准确识别正确读音。

## 🔧 扩展建议

如需扩展功能，可以：
1. 在 `src/data/hanzi.ts` 中添加更多汉字
2. 调整考试题目数量（修改 `startExam()` 参数）
3. 自定义配色方案（修改 `uno.config.ts`）
4. 添加音效或语音朗读功能
